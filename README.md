## About Laravel

This is for delivery managing service.
We will save the delivery service information in deliveries tables with service name, service api, and request parameters with "," string like "user_name,user_email..."

We will send request with the user details and product details and service id 
Then the delivery function in DeliveryController will send the request data to the selected service.
If we need to save the request data in table, we can create new table and will save it with status(if it is delivered to the selected service)
